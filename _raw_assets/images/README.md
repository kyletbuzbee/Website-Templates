## üéØ **How to Use This Guide**

### **Step 1: Choose Your Industry**
Pick the industry that matches your business and download those specific images.

### **Step 2: Download Images**
```bash
# Example: Download all fitness images
npm run bulk-download browser fitness

# Or download from stock photo sites:
# - Unsplash.com (free, high quality)
# - Pexels.com (free, high quality)
# - Pixabay.com (free, high quality)
# - Shutterstock.com (paid, premium)
```

### **Step 3: Save with Exact Names**
Save each downloaded image using the **exact filename** listed above. The automated pipeline expects these specific names.

### **Step 4: Process Images**
```bash
# Process all downloaded images
npm run distribute-assets

# Update templates to use optimized images
npm run update-templates

# Full production build
npm run build:full
```

### **Step 5: Professional Results**
Your templates now have:
- ‚úÖ Optimized images (WebP, AVIF, JPEG)
- ‚úÖ Responsive `<picture>` elements
- ‚úÖ Lazy loading
- ‚úÖ Perfect Core Web Vitals scores

## üìä **Image Specifications**

**Recommended Sizes:**
- **Hero images:** 1920x1080px (16:9 ratio)
- **Service images:** 800x600px
- **Team photos:** 600x600px (square)
- **Product images:** 800x800px (square)
- **Icons:** 256x256px (square)

**Quality Guidelines:**
- High resolution (2MB+ recommended)
- Well-lit, professional photography
- Consistent style and branding
- No watermarks or text overlays
- Diverse representation when applicable

**File Formats:**
- Save as: `.jpg`, `.jpeg`, or `.png`
- Pipeline converts to: `.webp`, `.avif`, `.jpeg` automatically
